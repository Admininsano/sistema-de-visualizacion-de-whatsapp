# WhatsApp Chat Visualizer

Sistema web para importar exportaciones `.txt` de WhatsApp, reconstruir la conversación y navegarla como un chat moderno.

## Stack

- Frontend: React + Vite + TailwindCSS
- Backend: Node.js + Express
- Parsing: TypeScript, una pasada sobre el archivo para soportar chats grandes

## Desarrollo

```bash
npm install
npm run dev
```

Frontend: `http://localhost:5173`

API: `http://localhost:4000`
